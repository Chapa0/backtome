# Cambios 0.0.5+5

Esta version inicia despues del cierre de `0.0.4+4`.

## Areas

- `actualizaciones_dialogo/`: correccion del disparo global del dialogo de
  actualizacion y limpieza de Ajustes para usuario final.
- `flujo_autenticacion/`: correccion del arranque para mostrar Login sin sesion
  y redirigir admin/usuario segun su rol.
- `flujo_admin_mapa/`: recuperacion de la home admin con mapa y `UP_PANEL`.
- `puntos_entrega_reclamacion/`: reemplazo del mapa estatico por puntos
  configurables de entrega/reclamacion, con custodia del objeto y recepcion en
  punto.
