# Visor de imagenes

## Objetivo

Traer a BackToMe el visor de imagenes usado en `aesantarita_app` para que las
imagenes de objetos perdidos puedan abrirse en pantalla completa.

## Implementacion esperada

- Crear un widget compartido para imagenes desde archivo, bytes o URL.
- Usar el visor en las imagenes principales de objetos perdidos.
- Usar el visor en imagenes de evidencia de reclamaciones.
- Mantener `pubspec.yaml` en `0.0.3+3`; no publicar `v0.0.3` hasta confirmar
  el release.
