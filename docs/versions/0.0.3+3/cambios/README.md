# Cambios 0.0.3+3

Esta version inicia despues del cierre de `0.0.2+2`.

## Areas

- `versionado/`: preparacion de la nueva version y ajustes para que Android use
  la version declarada por Flutter.
- `recuperacion/`: documentacion de la recuperacion despues de una mezcla
  accidental de codigo al hacer pull desde una version anterior hacia la rama
  principal.
- `visor_imagenes/`: visor reutilizable para abrir imagenes de objetos en
  pantalla completa con zoom y rotacion.
