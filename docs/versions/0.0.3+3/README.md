# 0.0.3+3

## Estado

Version lista para publicacion como actualizacion Android.

## Objetivo

Preparar la siguiente actualizacion sin interferir con el sistema de
actualizaciones ya publicado en `0.0.2+2`.

## Areas de cambio

| Area | Descripcion |
|---|---|
| versionado | Inicio de version, alineacion de version Android y reglas de actualizacion |
| recuperacion | Recuperacion del proyecto despues de una mezcla accidental con la rama principal |
| visor_imagenes | Visor reutilizable para abrir imagenes en pantalla completa |

## Notas de alcance

- Esta version corresponde a la siguiente actualizacion de la app.
- El tag de GitHub esperado al publicar es `v0.0.3`.
- El APK esperado debe corresponder a `version: 0.0.3+3`.
- El usuario confirmo la publicacion de esta version.
