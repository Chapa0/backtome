# Release 0.0.3+3

## Estado

Release final para publicar.

Esta version queda marcada como recuperacion despues del incidente de mezcla de
codigo ocurrido al hacer pull desde una version anterior. El problema ya fue
resuelto y la app queda lista para desplegar la version 3.

## Recuperacion del proyecto

Se corrigio el estado del proyecto despues de que una version anterior se
mezclara accidentalmente con la rama principal. La aplicacion vuelve a estar
alineada con la version correcta y lista para recibir actualizaciones desde
GitHub.

## Imagenes de objetos

Ahora las imagenes de objetos perdidos se pueden abrir en pantalla completa.
Desde el visor se puede acercar la imagen, moverla y rotarla para revisar mejor
los detalles.
