# 0.0.2+2

## Estado

Version en desarrollo activa para la siguiente actualizacion Android.

## Objetivo

Corregir el flujo de autenticacion, mejorar el panel inferior de la vista de
mapa y activar la experiencia de administracion dentro del mapa.

## Areas de cambio

| Area | Descripcion |
|---|---|
| autenticacion | Registro, inicio de sesion, sesion restaurada y validacion de contrasenas |
| mapa_admin | Vista de mapa para admin, filtros, aprobacion y gestion de entrega |
| up_panel | Panel inferior deslizable con gesto de arrastre |

## Notas de alcance

- Esta version corresponde a la siguiente actualizacion de la app.
- El tag de GitHub esperado al publicar es `v0.0.2`.
- El APK esperado debe corresponder a `version: 0.0.2+2`.
