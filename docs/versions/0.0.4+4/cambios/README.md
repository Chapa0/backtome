# Cambios 0.0.4+4

Esta version inicia despues del cierre de `0.0.3+3`.

## Areas

- `arranque_rutas/`: correccion de inicializacion de dependencias y registro de
  rutas de la app.
- `firestore_rules/`: configuracion para desplegar reglas de Firestore desde el
  repositorio.
- `sistema_actualizaciones/`: migracion del flujo de actualizaciones al modelo
  usado en `almeet`, con chequeo global, dialogos visibles y descarga/instalacion
  guiada.
