# Release 0.0.7+7

## Estado

Release publicado.

Release GitHub:
`https://github.com/Chapa0/backtome/releases/tag/v0.0.7`.

## Acciones mas claras

- Al aprobar, rechazar, entregar o registrar un objeto, la app muestra una
  pantalla de carga hasta recibir la respuesta.
- Mientras una accion esta en curso no es posible volver atras ni repetirla por
  accidente.
- La misma proteccion se aplica al administrar puntos, eliminar registros y
  realizar acciones importantes de la cuenta.

## Mapas que muestran donde ir

- El boton de ubicacion abre ahora un mapa interactivo, no una pantalla de
  coordenadas.
- El mapa indica con claridad donde se encontro el objeto y, cuando ya esta en
  custodia, el punto de entrega donde puede recogerse.
- Ambos lugares se muestran al mismo tiempo para evitar confusiones.

## Nuevos puntos de entrega

- Los puntos de entrega tienen un marcador turquesa con icono de establecimiento
  y ya no se confunden con los objetos perdidos.
- El punto relacionado con el objeto seleccionado se resalta para encontrarlo
  rapidamente.
